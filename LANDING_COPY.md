# FAQ

## 1. What does SpendPilot actually do?

SpendPilot helps users analyze their AI subscription spending and identifies possible optimization opportunities based on pricing plans, usage patterns, and team size.

---

## 2. Which AI tools are currently supported?

The current MVP supports tools like ChatGPT, Claude, Gemini, Cursor, GitHub Copilot, and Windsurf.

---

## 3. Does SpendPilot use real pricing data?

Yes. The application currently uses manually verified pricing references collected from the official pricing pages of the supported AI tools.

---

## 4. Are the recommendations AI-generated?

The financial calculations themselves are rule-based to keep them predictable and explainable. AI is mainly used to generate readable executive summaries from the audit results.

---

## 5. Do I need to create an account to use the tool?

No. Users can generate audits without authentication in the current MVP version.
